## AI Detection: How We Compare

### Our Approach

**Tformance:** Detects AI tool usage from PR metadata and patterns. Looks for Copilot, Cursor, Claude signatures in commits and PR descriptions. No code scanning required.

**Span:** Analyzes code patterns to identify AI-generated content. More granular detection at the code block level.
